# Simple typescript-nodejs template

```bash
 ~ git clone ts-node-basic
 ~ cd ts-node-basic
 ~ npm install
 ~ npm start
```

Please setup your .env file with the following var

~~~text
PORT=3000 || any port you want
~~~

## Available Scripts

~~~bash
    ~ npm run dev
~~~

Start developing your server things, with nodemon to cold reload after saving

~~~bash
    ~ npm run build
~~~

Compile your typescript files into js

~~~bash
    ~ npm run start
~~~

Build and run your node js app
